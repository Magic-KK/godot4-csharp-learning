# 🎮 Godot 4 C# 学习网站

一个完整的 Godot 4 C# 学习资源网站，涵盖从入门到精通的完整知识体系。

## 🌐 在线预览

访问：https://niuniu.github.io/godot4-csharp-learning

## 📚 内容结构

### 基础教程
- Godot 4 编辑器入门
- C# 语言基础回顾
- 节点与场景系统
- Godot 内置类型
- 输入处理系统
- 2D 游戏开发基础

### 进阶内容
- 信号（Signal）系统详解
- 资源（Resource）管理
- 高级动画系统
- 物理系统深入
- 设计模式在 Godot 中的应用
- 性能优化技巧
- 编辑器插件开发
- 网络与多人游戏

### 实战项目
- 2D 平台跳跃游戏
- 3D 第一人称探索
- 回合制战斗系统
- 完整 UI 系统实现
- 存档与数据管理系统
- 多人游戏演示

## 🛠️ 技术栈

- **Astro 5.x** - 现代静态站点生成器
- **TailwindCSS 4.x** - 实用优先的 CSS 框架
- **TypeScript** - 类型安全的 JavaScript 超集

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 📦 部署

本网站部署在 GitHub Pages：

```bash
# 构建
npm run build

# 推送 dist 目录到 gh-pages 分支
npx gh-pages -d dist
```

## 🎓 学习建议

1. **循序渐进** - 按照知识体系顺序学习，不要跳级
2. **动手实践** - 看完教程立即动手做，代码要自己敲
3. **记录笔记** - 建立自己的知识库，记录问题和解决方案
4. **完成项目** - 通过完整项目巩固所学知识
