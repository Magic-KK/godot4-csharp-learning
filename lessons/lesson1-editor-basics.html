Godot 4 编辑器入门

⏱️ 预计时间：30 分钟 · 难度：⭐

🎯 学习目标

  • ✓ 熟悉 Godot 4 的界面布局和各面板功能
  • ✓ 理解项目面板、场景面板、检查器的作用
  • ✓ 掌握文件系统管理和工作区切换
  • ✓ 记住常用快捷键，提高开发效率

📌 1. Godot 4 界面概览

Godot 4 的编辑器界面经过重新设计，更加现代化和直观。主界面分为以下几个核心区域：

📁 项目面板 (Project)

显示项目中的所有文件和资源，支持文件夹层级结构。可以拖拽文件到场景中快速创建节点。

🎬 场景面板 (Scene)

显示当前场景的节点树结构。可以添加、删除、重命名节点，以及调整节点层级关系。

⚙️ 检查器 (Inspector)

显示选中节点的属性和信号。可以修改属性值、连接信号、添加脚本。

👁️ 视图窗口 (Viewport)

2D/3D 编辑区域，可以预览和编辑场景内容。支持多视图布局和实时预览。

📌 2. 工作区切换

Godot 4 顶部工具栏提供了快速切换工作区的按钮，每个工作区针对不同的开发任务进行了优化：

2D

2D 游戏开发工作区，显示 2D 编辑工具和画布

3D

3D 游戏开发工作区，显示 3D 视图和建模工具

脚本

代码编辑工作区，支持 C# 和 GDScript

AssetLib

资源库，可以下载和安装社区创建的插件和资源

📌 3. 常用快捷键

快捷键 功能 工作区
Ctrl + N 创建新场景 通用
Ctrl + S 保存当前场景 通用
F5 运行项目 通用
F6 运行当前场景 通用
W / E / R 选择/移动/旋转工具 2D/3D
Ctrl + Shift + A 添加节点 场景
Ctrl + Alt + S 创建脚本 场景

📝 实践练习：创建第一个项目

练习目标

创建一个空的 Godot 4 项目，熟悉项目创建流程和界面布局。

步骤指南

  1. 1 启动 Godot 4，在项目管理器中点击「新建项目」
  2. 2 选择一个空文件夹作为项目目录，输入项目名称（如"MyFirstProject"）
  3. 3 选择渲染器（2D 项目推荐选择「兼容性」，3D 项目推荐「前进+）
  4. 4 点击「创建并编辑」，进入编辑器界面
  5. 5 浏览各个面板，尝试点击不同的工作区切换按钮
  6. 6 按 Ctrl + Shift + A 打开添加节点对话框，浏览可用的节点类型

💡 提示

不要担心「弄坏」什么，Godot 的项目结构很清晰，随时可以重新创建。大胆尝试是学习的最佳方式！
