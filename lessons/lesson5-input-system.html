📌 1. 输入映射配置

什么是输入映射？

Godot 的输入映射系统允许你将物理输入（按键、鼠标按钮等）映射到逻辑动作名称， 使代码更灵活、易于修改和重新绑定。

💡 优势

  • • 代码不直接绑定具体按键，易于修改
  • • 支持多按键映射到同一动作
  • • 方便实现按键自定义功能
  • • 统一处理键盘、鼠标、手柄

配置输入映射

在项目设置中配置输入映射：

步骤 1：打开项目设置

项目 (Project) → 项目设置 (Project Settings) → 输入映射 (Input Map)

步骤 2：添加动作

在「动作」栏输入动作名称，点击「添加」按钮：

move_up move_down move_left move_right jump attack

步骤 3：绑定按键

为每个动作添加按键（支持多个）：

move_up → W / ↑ / 手柄上
move_down → S / ↓ / 手柄下
move_left → A / ← / 手柄左
move_right → D / → / 手柄右
jump → Space / K / 手柄 A
attack → J / 鼠标左键 / 手柄 X

代码中动态添加映射

public override void _Ready() { // 动态添加输入映射（可选） if (!InputMap.HasAction("my_action")) { InputMap.AddAction("my_action"); InputMap.ActionAddEvent("my_action", new InputEventKey { Keycode = Key.Z }); } }

📌 2. Input 类常用方法

动作输入（推荐）

方法 说明 示例
IsActionPressed() 动作是否被按下（持续） Input.IsActionPressed("move_right")
IsActionJustPressed() 动作是否刚被按下（单次） Input.IsActionJustPressed("jump")
IsActionJustReleased() 动作是否刚被释放（单次） Input.IsActionJustReleased("attack")
GetActionStrength() 获取动作强度（0-1） float strength = Input.GetActionStrength("accelerate")

轴向输入（2D 移动）

// 获取轴向值（-1 到 1） float h = Input.GetAxis("left", "right"); // 水平轴 float v = Input.GetAxis("up", "down"); // 垂直轴 // 组合为向量 Vector2 direction = new Vector2(h, v); // 标准化（避免对角线更快） if (direction.LengthSquared() > 0) { direction = direction.Normalized(); } // 应用移动 velocity.X = direction.X * speed; velocity.Y = direction.Y * speed;

鼠标输入

// 鼠标位置 Vector2 mousePos = GetGlobalMousePosition(); // 2D Vector3 mousePos3D = GetGlobalMousePosition(); // 3D // 鼠标按钮 bool leftPressed = Input.IsMouseButtonPressed(MouseButton.Left); bool rightPressed = Input.IsMouseButtonPressed(MouseButton.Right); bool middlePressed = Input.IsMouseButtonPressed(MouseButton.Middle); // 鼠标滚轮 float wheelDelta = Input.GetAxis("ui_focus_prev", "ui_focus_next"); // 鼠标模式（FPS 游戏常用） Input.MouseMode = Input.MouseModeEnum.Captured; // 隐藏并锁定 Input.MouseMode = Input.MouseModeEnum.Visible; // 显示

手柄输入

📌 3. 实现角色控制器

2D 角色移动示例

using Godot; public partial class Player : CharacterBody2D { [Export] public float Speed { get; set; } = 200f; [Export] public float JumpVelocity { get; set; } = -400f; private float _gravity = ProjectSettings.GetSetting("physics/2d/default_gravity").AsSingle(); public override void _PhysicsProcess(double delta) { // 添加重力 if (!IsOnFloor()) { Velocity += new Vector2(0, _gravity) * (float)delta; } // 处理跳跃 if (Input.IsActionJustPressed("jump") && IsOnFloor()) { Velocity = new Vector2(Velocity.X, JumpVelocity); } // 处理移动 float inputDirection = Input.GetAxis("left", "right"); Velocity = new Vector2(inputDirection * Speed, Velocity.Y); // 应用移动 MoveAndSlide(); } }

8 方向移动（带动画）

public partial class Player : CharacterBody2D { [Export] public float Speed { get; set; } = 200f; private Sprite2D _sprite; private AnimationPlayer _animPlayer; public override void _Ready() { _sprite = GetNode("Sprite2D"); _animPlayer = GetNode("AnimationPlayer"); } public override void _PhysicsProcess(double delta) { // 获取输入方向 Vector2 direction = new Vector2( Input.GetAxis("left", "right"), Input.GetAxis("up", "down") ); // 标准化避免对角线更快 if (direction.LengthSquared() > 0) { direction = direction.Normalized(); _animPlayer.Play("walk"); // 翻转精灵 if (direction.X != 0) { _sprite.FlipH = direction.X < 0; } } else { _animPlayer.Play("idle"); } // 应用移动 Velocity = direction * Speed; MoveAndSlide(); } }

📝 实践练习：创建角色控制器

练习目标

创建一个完整的 2D 角色控制器，支持移动、跳跃和攻击。

练习要求

  1. 1 配置输入映射：move_left, move_right, jump, attack
  2. 2 创建 Player 场景（CharacterBody2D + Sprite + CollisionShape）
  3. 3 实现左右移动和跳跃功能
  4. 4 添加攻击动画和攻击判定
  5. 5 实现面向方向的翻转（FlipH）

💡 提示

使用 MoveAndSlide() 处理物理移动，它会自动处理碰撞和重力。
