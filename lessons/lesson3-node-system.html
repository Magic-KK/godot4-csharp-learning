📌 1. 节点树结构

Godot 的核心是节点树（Scene Tree）。一切皆是节点（Node），节点可以包含子节点，形成树状结构。

示例：一个简单游戏的节点树

Main (Node)
├── WorldEnvironment
├── Player (CharacterBody2D)
│   ├── Sprite2D
│   ├── CollisionShape2D
│   └── Camera2D
├── Enemy (CharacterBody2D)
│   ├── Sprite2D
│   └── CollisionShape2D
├── UI (Control)
│   ├── HealthBar
│   └── ScoreLabel
└── AudioManager (AudioStreamPlayer)

🌳 节点树的优势

  • • 清晰的层级结构
  • • 组件化设计
  • • 易于复用和维护
  • • 父子节点自动关联

💡 最佳实践

  • • 保持节点命名清晰
  • • 合理使用节点分组
  • • 避免过深的嵌套
  • • 使用场景实例化复用

📌 2. 场景的创建与实例化

创建场景

在 Godot 中，场景是一个保存的节点树。每个场景都有一个根节点。

方法 1：编辑器中创建

  1. 1. 点击场景面板的「新建场景」按钮
  2. 2. 选择根节点类型（Node2D、Node3D、Control 等）
  3. 3. 添加子节点并配置属性
  4. 4. 按 Ctrl+S 保存场景（.tscn 文件）

方法 2：代码中动态创建

// 创建节点 var player = new CharacterBody2D(); var sprite = new Sprite2D(); var collision = new CollisionShape2D(); // 建立层级关系 player.AddChild(sprite); player.AddChild(collision); // 添加到场景树 AddChild(player);

场景实例化

实例化允许你在场景中复用另一个场景：

public partial class Game : Node2D { [Export] public PackedScene EnemyScene { get; set; } private void SpawnEnemy(Vector2 position) { // 实例化场景 var enemy = EnemyScene.Instantiate(); enemy.Position = position; AddChild(enemy); } }

⚠️ 注意事项

  • • 使用 [Export] 让场景资源在编辑器中可分配
  • • Instantiate<T>() 指定返回类型
  • • 实例化后需要 AddChild() 添加到场景树
  • • 记得在不需要时调用 QueueFree() 释放

📌 3. 节点访问方法

方法 说明 示例
GetNode<T>() 获取子节点 GetNode<Sprite2D>("Player/Sprite")
GetParent<T>() 获取父节点 GetParent<Node2D>()
GetNodeOrNull<T>() 安全获取（可能为 null） GetNodeOrNull<AudioStreamPlayer>("SFX")
HasNode() 检查节点是否存在 HasNode("Player")
FindChild() 递归查找子节点 FindChild("Target", true, false)

路径语法

"Player" - 直接子节点
"Player/Sprite2D" - 后代节点（使用 / 分隔）
"../Enemy" - 兄弟节点（.. 表示父节点）
"/root/Game/Player" - 绝对路径（从根节点开始）
"~" - 当前节点

📌 4. 节点生命周期

Godot 节点按特定顺序调用生命周期方法：

_EnterTree()

节点进入场景树时调用

_Ready()

节点及其子节点都准备好后调用（只一次）

_Process(delta)

每帧调用（渲染帧）

_PhysicsProcess(delta)

固定间隔调用（物理帧，默认 60Hz）

_ExitTree()

节点离开场景树时调用

📝 实践练习：创建场景层次结构

练习目标

创建一个完整的游戏场景，包含玩家、敌人和 UI。

步骤指南

  1. 1 创建主场景 Main.tscn，根节点为 Node2D
  2. 2 创建玩家场景 Player.tscn（CharacterBody2D + Sprite + CollisionShape）
  3. 3 将 Player 场景实例化到 Main 场景中
  4. 4 创建 UI 场景（Control 节点 + Label）
  5. 5 在代码中使用 GetNode 访问子节点

💡 提示

使用场景面板的「+」按钮快速添加节点，按 F1 查看节点文档。
