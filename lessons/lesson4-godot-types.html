Godot 内置类型

⏱️ 预计时间：45 分钟 · 难度：⭐⭐

🎯 学习目标

  • ✓ 掌握 Godot 特有的数学类型（Vector2/3、Color 等）
  • ✓ 理解变换类型（Transform2D/3D、Quaternion）
  • ✓ 学会使用 Godot 的全局工具类（GD、Mathf）
  • ✓ 熟悉常用类型的实用方法

📌 1. 向量类型

Vector2 - 2D 向量

用于 2D 位置、方向、速度等。

// 创建向量 Vector2 position = new Vector2(100, 200); Vector2 direction = new Vector2(1, 0); // 向右 Vector2 up = Vector2.Up; // (0, -1) Godot 中 Y 轴向下为正 Vector2 zero = Vector2.Zero; // (0, 0) // 常用属性 float x = position.X; float y = position.Y; float length = position.Length; // 向量长度 Vector2 normalized = position.Normalized; // 单位向量 // 常用方法 Vector2 result = position + direction; // 向量加法 float distance = position.DistanceTo(new Vector2(0, 0)); Vector2 rotated = position.Rotated(Mathf.Pi / 4); // 旋转 45 度 float dot = position.Dot(direction); // 点积 float cross = position.Cross(direction); // 叉积（返回标量） // 插值 Vector2 lerpResult = position.Lerp(target, 0.5f); // 线性插值 Vector2 slerpResult = position.Slerp(target, 0.5f); // 球面插值

Vector3 - 3D 向量

用于 3D 位置、方向、速度等。

// 创建向量 Vector3 position = new Vector3(1, 2, 3); Vector3 forward = Vector3.Forward; // (0, 0, -1) Vector3 up = Vector3.Up; // (0, 1, 0) Vector3 right = Vector3.Right; // (1, 0, 0) // 常用属性 float x = position.X; float y = position.Y; float z = position.Z; Vector2 xy = position.Xy; // 提取 XY 平面 // 常用方法 float length = position.Length; Vector3 normalized = position.Normalized; Vector3 rotated = position.Rotated(Vector3.Up, Mathf.Pi / 4); float dot = position.Dot(other); Vector3 cross = position.Cross(other); // 返回向量 // 3D 特有 Vector3 floor = position.Floor(); Vector3 ceil = position.Ceil(); Vector3 snapped = position.Snapped(new Vector3(0.1f, 0.1f, 0.1f));

💡 实用技巧

移动角色

Vector2 velocity = Vector2.Zero; velocity.X = Input.GetAxis("left", "right") * speed; velocity.Y = Input.GetAxis("up", "down") * speed; velocity = velocity.Normalized() * speed;

朝向目标

Vector2 direction = (targetPosition - position).Normalized(); rotation = direction.Angle(); // 2D // 或 LookAt(targetPosition); // 3D

📌 2. 颜色与矩形

Color - 颜色

// 预定义颜色 Color red = Colors.Red; Color blue = Colors.Blue; Color white = Colors.White; Color transparent = Colors.Transparent; // 自定义颜色 Color custom1 = new Color(1, 0, 0); // RGB (0-1) Color custom2 = new Color(1, 0, 0, 0.5f); // RGBA Color custom3 = Color.FromHtml("#ff0000"); // HTML 色值 Color custom4 = Color.FromRgb(255, 0, 0); // RGB (0-255) // 属性 float r = color.R; float g = color.G;float b = color.B; float a = color.A; // 方法 Color lighter = color.Lightened(0.2f); Color darker = color.Darkened(0.2f); Color inverted = color.Inverted(); Color blended = color.Blend(overlayColor);

Rect2 - 2D 矩形

// 创建矩形 Rect2 rect1 = new Rect2(0, 0, 100, 100); // x, y, width, height Rect2 rect2 = new Rect2(new Vector2(0, 0), new Vector2(100, 100)); // 属性 float x = rect.Position.X; float y = rect.Position.Y; float w = rect.Size.X; float h = rect.Size.Y; float right = rect.End.X; // 右边缘 float bottom = rect.End.Y; // 下边缘 // 方法 bool contains = rect.HasPoint(new Vector2(50, 50)); Rect2 expanded = rect.Grow(10); // 扩展 10 像素 Rect2 intersect = rect.Intersect(other); bool overlaps = rect.Intersects(other);

📌 3. 变换类型

Transform2D - 2D 变换

包含旋转、缩放、平移信息。

// 创建变换 Transform2D identity = Transform2D.Identity; Transform2D transform = new Transform2D(); // 设置属性 transform.Origin = new Vector2(100, 100); // 平移 transform = transform.Rotated(Mathf.Pi / 4); // 旋转 45 度 transform = transform.Scaled(new Vector2(2, 2)); // 缩放 2 倍 // 组合变换 Transform2D combined = transform1 * transform2; // 应用变换 Vector2 transformedPoint = transform * localPoint; // 在节点中使用 transform = GlobalTransform; // 获取全局变换 GlobalTransform = transform; // 设置全局变换

Transform3D - 3D 变换

// 创建变换 Transform3D identity = Transform3D.Identity; Transform3D transform = new Transform3D(); // 设置属性 transform.Origin = new Vector3(0, 0, 0); transform.Basis = Basis.Identity; // 使用 Basis 进行旋转缩放 transform.Basis = transform.Basis.Rotated(Vector3.Up, angle); transform.Basis = transform.Basis.Scaled(scale); // 快速方法 transform = transform.Translated(offset); transform = transform.Rotated(axis, angle); transform = transform.Scaled(scale); // 在 3D 节点中使用 GlobalTransform = transform; Transform3D current = GlobalTransform;

Quaternion - 四元数

用于 3D 旋转，避免万向节锁。

// 创建四元数 Quaternion identity = Quaternion.Identity; Quaternion rotation = new Quaternion(axis, angle); // 从欧拉角创建 Quaternion q = Quaternion.FromEuler(new Vector3(pitch, yaw, roll)); // 常用方法 Quaternion result = q1 * q2; // 组合旋转 Vector3 rotated = q * vector; // 旋转向量 Quaternion inverted = q.Inverse(); // 插值（平滑旋转） Quaternion slerpResult = Quaternion.Slerp(q1, q2, 0.5f); // 转换为欧拉角 Vector3 euler = q.GetEuler();

📌 4. 全局工具类

GD - Godot 全局工具

// 输出调试 GD.Print("Hello, Godot!"); GD.Print($"Position: {position}"); GD.PrintErr("Error message!"); GD.PrintStack(); // 打印调用栈 // 数学工具 float result = GD.RandRange(0, 10); // 随机数 int intResult = GD.RandiRange(0, 10); Vector3 randomDir = GD.RandomUnitVector(); // 类型转换 int num = GD.StrToInt("123"); string str = GD.IntToStr(123); Variant variant = GD.VarToString(value); // 时间与帧 double time = GD.GetProcessTime(); int fps = Engine.GetFramesPerSecond(); // 其他实用方法 string path = GD.ProjectSettingsGlobalize("res://"); bool fileExists = GD.FilesystemHas("res://file.txt");

Mathf - 数学工具

// 常量 float pi = Mathf.Pi; // 3.14159... float tau = Mathf.Tau; // 2 * Pi float inf = Mathf.Inf; float nan = Mathf.NaN; // 三角函数 float sin = Mathf.Sin(angle); float cos = Mathf.Cos(angle); float tan = Mathf.Tan(angle); float angle2 = Mathf.Atan2(y, x); // 常用运算 float sqrt = Mathf.Sqrt(16); float pow = Mathf.Pow(2, 3); float abs = Mathf.Abs(-5); float sign = Mathf.Sign(-5); // -1 // 取整 float floor = Mathf.Floor(3.7f); // 3 float ceil = Mathf.Ceil(3.2f); // 4 float round = Mathf.Round(3.5f); // 4 // 插值 float lerp = Mathf.Lerp(0, 10, 0.5f); // 5 float angleLerp = Mathf.LerpAngle(a1, a2, 0.5f); // 限制 float clamped = Mathf.Clamp(value, 0, 100); float snapped = Mathf.Snapped(value, 0.1f);
