📌 1. 2D 精灵与纹理

Sprite2D - 2D 精灵

用于显示 2D 图像，是最常用的 2D 节点。

// 在编辑器中设置纹理 [Export] public Texture2D PlayerTexture { get; set; } public override void _Ready() { var sprite = GetNode("Sprite2D"); sprite.Texture = PlayerTexture; // 常用属性 sprite.FlipH = true; // 水平翻转 sprite.FlipV = true; // 垂直翻转 sprite.Modulate = Colors.Red; // 颜色调制 sprite.Visible = true; // 可见性 }

AnimatedSprite2D - 动画精灵

用于播放精灵动画，需要 SpriteFrames 资源。

public partial class Player : CharacterBody2D { private AnimatedSprite2D _animatedSprite; public override void _Ready() { _animatedSprite = GetNode("AnimatedSprite2D"); } public override void _PhysicsProcess(double delta) { Vector2 direction = new Vector2( Input.GetAxis("left", "right"), Input.GetAxis("up", "down") ); if (direction.LengthSquared() > 0) { direction = direction.Normalized(); _animatedSprite.Play("walk"); // 根据方向翻转 if (direction.X < 0) _animatedSprite.FlipH = true; else if (direction.X > 0) _animatedSprite.FlipH = false; } else { _animatedSprite.Play("idle"); } } }

⚙️ 创建动画步骤

  1. 1. 选中 AnimatedSprite2D 节点
  2. 2. 在检查器中点击「新建 SpriteFrames」
  3. 3. 点击「SpriteFrames」面板
  4. 4. 拖拽图片到动画帧区域
  5. 5. 设置 FPS 和循环选项

TextureRect - 纹理矩形

用于 UI 中的图片显示，支持拉伸模式。

// 常用拉伸模式 var textureRect = GetNode("TextureRect"); textureRect.StretchMode = TextureRect.StretchModeEnum.KeepAspect; // 其他模式：Scale、Keep、KeepCover、KeepCentered // 设置纹理 textureRect.Texture = someTexture; textureRect.ExpandMode = TextureRect.ExpandModeEnum.ExpandToFit;

📌 2. TileMap 地图系统

什么是 TileMap？

TileMap 是一种高效的 2D 地图系统，使用小图块（Tile）拼接成大地图。 适用于平台游戏、RPG、策略游戏等。

✅ 优势

  • • 内存效率高
  • • 绘制性能好
  • • 易于编辑和修改
  • • 支持自动平铺

🎮 适用场景

  • • 平台跳跃游戏地面
  • • RPG 世界地图
  • • 策略游戏地形
  • • 迷宫/地牢生成

配置 TileMap

步骤 1：创建 TileSet

  1. 1. 选中 TileMap 节点
  2. 2. 在检查器中点击「新建 TileSet」
  3. 3. 点击 TileSet 属性旁的箭头进入编辑

步骤 2：添加图块纹理

  1. 1. 在底部面板选择「图块集」选项卡
  2. 2. 点击「+」添加新图块
  3. 3. 拖拽纹理到预览区域
  4. 4. 设置图块大小（如 16x16、32x32）

步骤 3：绘制地图

  1. 1. 选择 TileMap 节点
  2. 2. 点击工具栏的「绘制」工具
  3. 3. 在场景中选择图块并绘制
  4. 4. 使用不同图层绘制地面、墙壁、装饰

代码中操作 TileMap

public partial class LevelManager : Node2D { private TileMap _tileMap; public override void _Ready() { _tileMap = GetNode("TileMap"); } // 设置指定位置的图块 public void SetTile(Vector2I position, int tileId) { _tileMap.SetCell(0, position, 0, new Vector2I(tileId, 0)); } // 清除指定位置的图块 public void ClearTile(Vector2I position) { _tileMap.SetCell(0, position, -1); } // 获取指定位置的图块 public Vector2I GetTile(Vector2I position) { return _tileMap.GetCellSourceId(0, position); } }

📌 3. Camera2D 摄像机

跟随玩家

Camera2D 用于控制游戏视图，常用于跟随玩家角色。

// 简单方法：在编辑器中设置 // 1. 添加 Camera2D 作为 Player 的子节点 // 2. 勾选「当前」(Current) 属性 // 3. 摄像机自动跟随玩家移动

代码控制摄像机

public partial class GameCamera : Camera2D { [Export] public Node2D Target { get; set; } [Export] public float SmoothSpeed { get; set; } = 5f; [Export] public Vector2 Offset { get; set; } = Vector2.Zero; private Vector2 _velocity; public override void _PhysicsProcess(double delta) { if (Target == null) return; // 平滑跟随 Vector2 targetPosition = Target.Position + Offset; Position = Position.Lerp(targetPosition, (float)delta * SmoothSpeed); // 或者使用插值 // Position = Position.InterpolateWith(targetPosition, (float)delta * SmoothSpeed); } }

摄像机属性

属性 说明 示例值
当前 (Current) 是否为活动摄像机 true
缩放 (Zoom) 摄像机缩放倍率 Vector2(1, 1)
旋转 (Rotation) 摄像机旋转角度 0
限制 (Limit) 摄像机移动边界 Left, Top, Right, Bottom
平滑 (Smooth) 启用平滑跟随 true

📌 4. 碰撞检测

碰撞形状节点

CollisionShape2D

定义碰撞区域形状

支持：Rectangle, Circle, Capsule, Polygon

CollisionPolygon2D

自定义多边形碰撞

适用于不规则形状

碰撞检测信号

public partial class Player : CharacterBody2D { public override void _Ready() { // 连接碰撞信号 var collisionShape = GetNode("CollisionShape2D"); // 注意：信号从 Area2D 或 CharacterBody2D 发出 } // CharacterBody2D 的碰撞处理 public override void _PhysicsProcess(double delta) { MoveAndSlide(); // 检查碰撞 for (int i = 0; i < GetSlideCollisionCount(); i++) { var collision = GetSlideCollision(i); var collider = collision.GetCollider(); if (collider is Enemy enemy) { TakeDamage(enemy.Damage); } } } private void TakeDamage(int amount) { GD.Print($"Player took {amount} damage!"); } }

Area2D 检测区域

public partial class Pickup : Area2D { public override void _Ready() { // 连接信号 BodyEntered += OnBodyEntered; } private void OnBodyEntered(Node2D body) { if (body is Player) { // 玩家拾取物品 GD.Print("Item collected!"); QueueFree(); // 删除物品 } } } // 或者在检查器中连接信号 // area2d → body_entered → 选择处理方法
